# Quiz Arena — квиз-платформа реального времени

Веб-приложение для проведения квизов в реальном времени (аналог Kahoot).
Учебный проект. Две роли: **организатор** создаёт и ведёт квиз, **участник**
подключается по коду комнаты и отвечает на вопросы по таймеру.

## Возможности

- Регистрация и вход, роли организатора и участника
- Конструктор квизов: категории, время на вопрос, картинки по ссылке
- Типы вопросов: один правильный ответ и несколько правильных
- Запуск игры → участники входят по коду комнаты
- Показ вопросов всем одновременно; ответы принимаются только пока идёт таймер
  (проверка времени на сервере)
- Очки за скорость ответа и живой лидерборд
- Личный кабинет с историей: проведённые квизы (у организатора) и участие (у участника)

## Технологии

| Слой | Выбор |
|------|-------|
| Фреймворк | Next.js (App Router, JavaScript) |
| Реальное время | Socket.IO (WebSocket) |
| База данных | SQLite через встроенный `node:sqlite` (без нативной сборки) |
| Стили | Tailwind CSS |
| Пароли | bcryptjs |

Реалтайм требует постоянно живущий процесс, поэтому используется кастомный сервер
[`server.js`](server.js), запускающий Next.js и Socket.IO на одном порту.

## Запуск

```bash
npm install     # установить зависимости
npm run dev     # запустить локально → http://localhost:3000
```

## Структура

```
├─ server.js          # кастомный сервер: Next + Socket.IO
├─ app/               # страницы (главная, вход, кабинет, конструктор, игра)
├─ components/        # общие компоненты и иконки
├─ lib/               # db, auth, quizzes, game, history
├─ socket/handlers.js # логика игры (комнаты, вопросы, очки)
└─ data/quiz.db       # база SQLite (создаётся автоматически, в git не хранится)
```

## Как проверить игру

1. Окно 1 — организатор запускает квиз и получает код комнаты.
2. Окно 2 (инкогнито) — `/join`, ввод кода и имени.
3. Организатор нажимает «Начать игру» и ведёт по вопросам.
